h1. adstream-js-frameworks

This repository contains two Javascript libraries developed and maintained by [[Adstream Holdings|http://adstream.com]] to serve as a foundation technology for its Web applications released into production from year 2011 onward. The code in this repository is available under [[the Modified BSD License|License]].

Please refer to the [[Wiki pages|wiki]] for detailed documentation or peruse the [[Booze Shack|tree/master/examples/booze]] example project to quickly acquaint yourself with the libraries.
